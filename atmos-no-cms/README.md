## Overview

Atmos-No-CMS is a modern web application built with React and TypeScript, designed for the development of Atoms LED brand website. It leverages Vite for fast builds and hot module replacement, ensuring a smooth development experience. The project includes comprehensive configurations for build, type safety, and deployment on Vercel, streamlining the entire workflow. With a rich library of reusable components—such as pages, navigation, product cards, and visual effects—Atmos-No-CMS empowers developers to create scalable, maintainable, and dynamic websites efficiently.

**Why Atmos-No-CMS?**

This project simplifies complex frontend development by providing a robust architecture and ready-to-use components. The core features include:

- 🎨 **Fast Development:** Vite setup with React support for rapid iteration and hot module replacement.
- 🔧 **Type Safety:** TypeScript configurations ensuring modern JavaScript features and reliable code.
- 🚀 **Seamless Deployment:** Vercel integration for effortless build and deployment workflows.
- 🧩 **Reusable Components:** Modular UI elements like pages, sections, and visual effects for quick assembly.
- 🌟 **Enhanced UX:** Utility hooks and animated components to deliver smooth, engaging user experiences.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** TypeScript
- **Package Manager:** Npm

### Installation

Build Atmos-No-CMS from the source and install dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/itsrichardhaar/Atmos-No-CMS
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd Atmos-No-CMS
    ```

3. **Install the dependencies:**

**Using [npm](https://www.npmjs.com/):**

```sh
❯ npm install
```

### Usage

**Navigate to the project root:**

```sh
❯ cd atmos-no-cms
```

Run the project with:

**Using [npm](https://www.npmjs.com/):**

```sh
npm run dev
```

### Testing

Atmos-no-cms uses the {__test_framework__} test framework. Run the test suite with:

**Using [npm](https://www.npmjs.com/):**

```sh
npm test
```
